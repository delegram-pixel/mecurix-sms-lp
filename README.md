## Getting started

To get started with this project, run:

```bash
git clone https://github.com/Abdullah-988/penwwws-frontend
```

Setup `.env` file:

```
NEXT_PUBLIC_BACKEND_URL=""

GOOGLE_CLIENT_ID=""

NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET=""
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=""
```

then run:

```bash
npm install
```

and

```bash
npm run dev
```
